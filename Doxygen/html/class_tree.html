◆ findNextByte()

Node Tree.findNextByte ( byte  B)
private

Funcion auxiliar de la funcion "progressive_find".

Se busca un nodo con id = B entre los hijos de last_visited_node.

  • Si este existe se retorna su contenido index.
  • Si este no existe, se crea e inserta un nodo con id = B, index = next_index y una lista de hijos vacia; y se retorna index.

Precondicion: Haber realizado una busqueda almenos una vez, para que last_visited_node haga referencia a algun nodo. Postcondicion: El Nodo buscado pasa a ser el ultimo nodo visitado (last_visited_node).

Parámetros
BByte identificador (id) del nodo que se quiere buscar.
Devuelve
Nodo buscado entre los hijos de last_visited_node con id = B.

◆ progressive_find()

int Tree.progressive_find ( byte  B,
boolean  restart 
)

El funcionamiento varia segun el valor de restart:

  • Si restart == true: La busqueda se lleva a cabo sobre los hijos de la raiz del arbol.
  • Si restart == false: La busqueda se lleva a cabo sobre los hijos del ultimo nodo visitado.

Se busca un nodo con id = B.

  • Si este existe se retorna su contenido index.
  • Si este no existe, se crea e inserta un nodo con id = B, index = next_index y una lista de hijos vacia; y se retorna index.
Parámetros
BByte identificador (id) del nodo que se quiere buscar.
restartIndica si se debe realizar la busqueda sobre los hijos de la raiz del arbol (true) o obre los hijos del ultimo nodo visitado (false).
Devuelve
Retorna el "index" del nodo buscado. Si este no existia, index = next_index.
